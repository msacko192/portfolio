# Logo Brainstorm — DJEXA

## Brand Analysis

- **Nom** : DJEXA — court, percutant, le **X** est un ancre visuelle forte
- **Métier** : Studio de développement logiciel sur mesure (applications métier)
- **Cible** : Décideurs d'entreprises françaises
- **Vibe** : Moderne & Technique — inspiration Linear (sharp, minimal, premium)
- **Style choisi** : Symbole + Wordmark

## Ancres visuelles

Le **X** final de DJEXA est le point d'entrée le plus fort : symétrique, mémorable, scalable au favicon.
Le **D** est l'ouverture — plus subtil, convient à un lettermark.
L'abstraction géométrique (vitesse, modules, nœuds) représente le cœur métier : logiciels précis, sur mesure.

---

## 20 Variations

### Groupe 1 — Symbole X (logos 1–5)
L'X comme marque principale. Direct, fort, différenciant.
1. **Bold X** — bras épais, pointes nettes. Candidat principal.
2. **Thin X** — bras fins, élégant, Linear-style.
3. **X + cercle** — X inscrit dans un anneau circulaire.
4. **X inversé** — X blanc découpé dans un cercle noir (evenodd).
5. **X trait** — deux lignes fines se croisant. Ultra-minimal.

### Groupe 2 — Lettermark D (logos 6–9)
Le D seul, traité géométriquement.
6. **D classique** — D creux, courbes, proportions équilibrées.
7. **D barre large** — barre gauche très épaisse, arc fin. Personnalité affirmée.
8. **D angulaire** — polygone, zéro courbe. Technique, structuré.
9. **D inversé** — D blanc dans un carré arrondi noir. App-icon ready.

### Groupe 3 — Géométrie abstraite (logos 10–15)
Métaphores visuelles du logiciel sur mesure.
10. **Carrés imbriqués** — cadre + carré central. Précision, focus.
11. **3 barres décroissantes** — modules hiérarchiques. Stacks.
12. **3 barres diagonales** — vitesse, mouvement. Inspiration Linear.
13. **Triangle nœuds** — 3 cercles connectés. Systèmes, architecture.
14. **Arcs concentriques** — deux arcs + point central. Ciblage, précision.
15. **Flèche droite** — avant, décision. Simple et direct.

### Groupe 4 — Combiné / Expérimental (logos 16–20)
16. **X dans carré noir** — X blanc en fond sombre. Inverse du #1.
17. **Chevrons `< >`** — code-style. Identity développeur-native.
18. **Orbital** — anneau + planète + centre. Système en orbite.
19. **Double chevron `>>`** — mouvement vers l'avant, itération.
20. **Grille de points D** — D en matrice de points. Craft + précision.
